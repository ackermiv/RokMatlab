Vorwärtskinematik:

1) Öffnen des Matlab Scripts: Vorwaertskinematik_Horst_HMI.m
2) Script ausführen (Dateipfade werden automatisch zum Current Folder hinzugefügt) 
3) Mit dem Button "Horst Anzeigen" Wird der Roboter in siener Grundposition dargestellt.
4) Mit den Schiebereglern, kann jede Achse einzeln verfahren werden. (Achswinkel werden zusätzlich digital dargestellt)

Inverse Kinematik:
1) InverseKinematik_Horst ausfuehren
    - zeigt den bewegten roboter
    - zeigt den absolvierten (rot) im vergleich zum vorgesehen pfad (schwarz)
